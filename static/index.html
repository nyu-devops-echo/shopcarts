Shopcart Demo REST API Service

GET /shopcarts to list all shopcarts.

GET /shopcarts/{id} retrieves a shopcart from the DB using an ID.

PUT /shopcarts/{uid}/products/{pid} updates a shopcart using an ID.

DELETE /shopcarts/{id} removes a shopcart from the DB using an ID.

POST /shopcarts creates a new shopcart.
